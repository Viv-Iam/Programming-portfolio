ABOUT ME


Back-end developer || With a passion for developing innovative mobile applications

I am passionate about machine control. The career path I chose is Aerospace/Mechanical Engineering which became focused in Industrial Internet of Things. I joined Moringa school (a world-class software developer accelerator based in Nairobi, Kenya) with the aim of learning how to develop systems that can be used to monitor, control, and visualize every aspect of operations for industrial companies. Currently I am developing innovative re-usable mobile applications that can be customized to the users needs.

PORT FOLIO

MOBILE-CHEKCKIN

MobileCheck-in is an Android application that allows users to remotely clock in at the click of a button. It automatically logs the date and time the user checks-in.

TATU SAFETY

Tatu Safety is an Android application that aims to ease travel via Public Transport. It also avails a platform for reporting malpractices.

TECH₂0



TecH₂O is a web app that utilizes satellite and crowd-sourced data to avail realtime information on fluctuating water levels across the globe. This app was awarded Local Peoples' Choice Award for the NASA SpaceApps Challenge 2017.

TRANSPARENCY
