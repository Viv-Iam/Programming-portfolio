ABOUT ME


Back-end developer||With a passion for developing innovative mobile applications

I am passionate about machine control. The career path I chose is Aerospace/Mechanical Engineering which became focused in Industrial Internet of Things. I joined Moringa school (a world-class software developer accelerator based in Nairobi, Kenya) to solely learn how to develop systems that can be used to monitor, control, and visualize every aspect of operations for industrial companies. Currently I am developing innovative re-usable mobile applications that can be customized to the users needs.

PORTFOLIO

MobileCheck-in is an Android application that allows users to clock in at the click of a button. It automatically logs the date and time the user checks-in.

Tatu Safety is an Android application that aims to ease travel via Public Transport. It also avails a platform for reporting malpractices.

TECH₂0



TecH₂O is a web app that utilizes satellite and crowd-sourced data to avail realtime information on fluctuating water levels across the globe. The app was awarded Local Peoples' Choice Award for the NASA SpaceApps Challenge 2017.

E-CHECKIN


E-checkin is an electronic web application, that allows individuals of particular institutions/organizations to check-in via an electronic system instead of the traditional paper sign-in. On click of check-in button it logs the date and time the user clocked in.

TRANSPARENCY


Transparency is a simple web application that encourages transparency and improves record-keeping in service-based businesses.
